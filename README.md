# Async Herald

**Améliorez la qualité de votre code en un clic.**

Async Herald analyse votre projet et vous donne un score de qualité avec des recommandations concrètes pour améliorer votre code.

---

## Pourquoi utiliser Async Herald ?

- **Simple** : Un clic pour analyser tout votre projet
- **Rapide** : Résultats en quelques secondes
- **Actionnable** : Des conseils clairs pour corriger chaque problème
- **Complet** : Analyse la sécurité, les performances, la maintenabilité et plus encore

---

## Comment ça marche ?

### 1. Connectez-vous

Ouvrez le panneau Async Herald dans la barre latérale et cliquez sur **"Lier mon compte"**.

### 2. Lancez l'analyse

Cliquez sur **"Analyser le projet"**. Herald scanne automatiquement vos fichiers de code.

### 3. Consultez vos résultats

Vous obtenez :
- Un **score global** sur 100
- Une **note** (A+, A, B+, etc.)
- La liste des **problèmes détectés** avec des suggestions de correction

---

## Ce qu'Herald analyse

Herald examine votre code selon 6 dimensions :

| Dimension | Description |
|-----------|-------------|
| **Sécurité** | Vulnérabilités, injections, failles potentielles |
| **Performance** | Code lent, requêtes inefficaces, fuites mémoire |
| **Maintenabilité** | Code complexe, duplication, dette technique |
| **Fiabilité** | Bugs potentiels, erreurs de logique |
| **Bonnes pratiques** | Conventions, patterns, standards du langage |
| **Documentation** | Code non documenté, commentaires manquants |

---
